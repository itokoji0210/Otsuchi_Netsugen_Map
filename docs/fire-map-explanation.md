# 大槌町 火災熱源マップについて

このマップは、岩手県大槌町周辺で観測された熱源データを、時間・場所・地形の文脈とあわせて確認するための可視化です。公開データを見やすく整理した補助資料であり、被害状況、焼失範囲、延焼範囲、避難の要否を示すものではありません。

## 表示しているもの

中心となるデータは、NASA FIRMSが公開している衛星観測にもとづく熱異常検知データです。各点には、観測時刻、緯度・経度、センサー種別、FRP、信頼度などの属性があります。

FRPはFire Radiative Powerの略で、衛星が観測した熱放射の強さを表す指標です。ただし、FRPが高い点がそのまま火災の規模や危険度を表すわけではありません。衛星が観測した画素内の熱的な特徴を示す値として扱ってください。

背景には国土地理院の写真タイルと地図タイルを使用しています。写真は現地の地形や土地利用の文脈を読み取るための背景であり、火災発生時点のリアルタイム画像ではありません。

## 熱源点の見方

熱源は、ホタルの光のような小さな発光点で表示しています。これは観測点を見つけやすくするための表現で、光の広がりが延焼範囲や被害範囲を示すものではありません。

新しい観測点や強い熱源ほど目立ちやすく表示していますが、これは時系列の分布を追いやすくするための視覚表現です。点がない場所に火がない、点がある場所が必ず燃えている、という意味ではありません。

既定では、観測開始から選択時刻までの熱源点を累積で表示します。これは、過去に観測された点が消えることで「収束した」「鎮火した」と誤解されることを避けるためです。

「直近6時間」「直近24時間」は、時間幅を絞って観測の変化を確認するための分析用モードです。直近表示で見えない点が、安全になった場所や消火済みの場所を意味するわけではありません。

## 風向・風速について

風向レイヤーは、Open-MeteoのJMA APIからJMA MSMモデルの10m高度の風向・風速を取得して表示します。これは現地観測所のリアルタイム実測値ではなく、数値予報モデルにもとづく推定・予報値です。JMA MSMは日本周辺を対象とする数値予報モデルで、Open-Meteoの説明ではおおむね5km解像度、1時間値、4日先までの予報、3時間ごとの更新です。

矢印は、熱源点が観測された時刻の前後に対応するモデル値を参照して表示します。風速が大きいほど矢印は長く・太くなり、色も青から水色、黄色、橙色へ変化します。これは延焼を予測するものではなく、観測された熱源分布と、その時刻周辺の風の場を見比べるための補助情報です。

モデル値は局所的な谷風・斜面風・突風をそのまま表すものではありません。通信やAPIの状態によってOpen-Meteoから取得できない場合は、画面上の風向表示が固定の参考値に切り替わることがあります。

## 読み取り時の注意

衛星観測には、観測時刻の間隔、雲や煙、地形、センサーの分解能、誤検知・未検知などの制約があります。表示される熱源点は、実際の火災の境界線や焼失面積を直接示すものではありません。

このマップは、公式な避難情報、消防・自治体の発表、現地の安全確認に代わるものではありません。安全に関わる判断は、必ず自治体、消防、警察、気象機関などの公式情報を確認してください。

## データと出典

- 熱源データ: NASA FIRMS
- 風向・風速: Open-Meteo JMA API / JMA MSM
- 背景地図・写真: 国土地理院 地理院タイル
- 表示ライブラリ: CesiumJS

地図や画像を引用・掲載する場合は、少なくとも上記の出典を併記してください。
