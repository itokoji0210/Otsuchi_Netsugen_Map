# News Tip Search Pack

調査日: 2026-04-27

## 使い方

1. 直近24-72時間は `recent_core` と `social_signal` を先に見る。
2. 翌日以降1か月以内の現場は `near_future_event` と `official_confirmation` で裏取りする。
3. SNSで見つけたネタは、必ず公式サイト、自治体、施設、主催者、PR配信、地方紙・地方局で確認する。
4. 候補化する条件は「現場が撮れる」「時事ワードと接点がある」「日付と問い合わせ先が確認できる」の3点。
